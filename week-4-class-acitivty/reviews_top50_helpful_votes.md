# `reviews_top50_helpful_votes.md`

## Purpose

CSV produced by `fetch_reviews.py`: the **50** review rows with the **largest** `helpful_votes` values (ties broken by sort order among equal votes).

## Inputs / dependencies

- Generated by running `python3 fetch_reviews.py` (requires a successful API fetch first).

## Outputs

- **`reviews_top50_helpful_votes.csv`** — columns `category`, `helpful_votes`, sorted with highest votes first.
